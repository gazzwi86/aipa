AIPA starts cold and wakes in ~45 seconds when you access it. Auto-shuts down after 30 minutes of idle time.

Quick Start

# Clone the repository
git clone https://github.com/gazzwi86/aipa.git
cd aipa

# Configure environment
cp .env.example .env
claude setup-token  # Get OAuth token for Claude

# Run locally
docker compose -f docker-compose.yml up --build

# Open http://localhost:8000

Add Agents

Define specialized agents in .claude/agents/ for different types of tasks.

Connect Tools
