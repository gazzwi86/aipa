# Deep Research Skill

Conduct thorough research on topics and produce comprehensive reports.

## When to Activate

- "Research thoroughly about..."
- "Write a comprehensive report on..."
- "Deep dive into..."
- "Investigate and report on..."
- "I need detailed research on..."

## Capabilities

### Multi-Source Research
- Web searches across multiple queries
- Documentation lookup
- Academic-style source evaluation
- Cross-referencing findings

### Report Generation
- Executive summaries
- Detailed findings
- Source citations
- Recommendations
- Appendices

### Analysis Types
- Comparative analysis
- Trend analysis
- SWOT analysis
- Risk assessment
- Cost-benefit analysis

## Research Process

### 1. Understand the Question
```
- What exactly needs to be answered?
- What's the scope (broad overview vs specific details)?
- Who is the audience?
- What format is expected?
```

### 2. Search Strategy
```
- Start with broad searches to understand landscape
- Narrow to specific queries for details
- Use multiple search engines/sources
- Look for primary sources
```

### 3. Source Evaluation
```
Priority order:
1. Official documentation
2. Academic papers / studies
3. Government / institutional sources
4. Industry experts
5. Reputable publications
6. Community sources (verify carefully)
```

### 4. Synthesis
```
- Identify common themes
- Note contradictions
- Highlight consensus
- Flag uncertainties
```

### 5. Report Writing
```
- Lead with key findings
- Support with evidence
- Cite all sources
- Acknowledge limitations
```
